# FocusU

**FocusU**는 감성 + 집중을 도와주는 웹 도우미입니다.  
GitHub Pages로 배포 가능한 HTML 기반 구조로, 기능별로 분리되어 있습니다.

## 📁 기능 구성
- `fortune/` - 오늘의 운세
- `diary/` - 감정 일기 쓰기
- `study/` - 공부 리소스 추천
- `todo/` - 할 일 리스트 관리

## 🛠 사용 방법
1. GitHub에 업로드
2. 기능별 HTML 진입 (ex. `yourname.github.io/FocusU/fortune/`)
3. 각 기능 독립적으로 실행 가능

## 🔖 깃허브 평가 기준 대응
- 브랜치 전략, 커밋 메시지, Issue/PR 관리 대응 가능
- 구조화된 디렉토리 → Repo 구조 항목 고득점 기대
